# Retro Engineer Workshop Portfolio

This is a personal portfolio website for Mohamed Diaa, designed with an "Old Engineer's Workshop" / FNAF aesthetic.

## features
- **Theme**: Rusty, worn workshop with sticky notes and blueprints.
- **Interactive Elements**: Typing effects, flickering lights, gear animations.
- **Sections**:
  - **Landing**: Intro on sticky notes.
  - **Past Work**: Filing cabinet style display of projects.
  - **Certifications**: Gallery of certificates on the wall.
  - **Contact**: Interactive paper form.

## How to View
Simply open `index.html` in your web browser.

## File Structure
- `index.html`: Main entry point.
- `css/style.css`: All conflicting styles and animations.
- `js/script.js`: Logic for tabs, data rendering, and particles.
- `assets/`: 
  - `img/`: Generated textures and backgrounds.
  - `Certifications/`: Your certification images.
  - `PastWorkContent/`: Other assets.
